## 1. 「2040年問題」とは何か

**2040年問題**とは、団塊ジュニア世代（1971〜1974年生まれ）が65歳以上を迎える2040年前後に、日本社会が直面する複合的な危機の総称である。

### 主な予測データ

| 指標 | 現状（2020年） | 2040年予測 |
|------|---------------|-----------|
| 日本の総人口 | 約1億2,615万人 | **約1億1,284万人**（▲1,330万人） |
| 高齢者（65歳以上）割合 | 約29% | **約35%** |
| 現役世代の支え比率 | 約2人で1人 | **1.5人で1人** |
| 労働力不足 | — | **約1,100万人不足** |
| 社会保障給付費 | 121兆円（2018年） | **約190兆円（1.6倍）** |

### 社会保障費の内訳

- 介護費：**2.4倍**に膨張
- 医療費：**1.7倍**に膨張
- 年金：約1.3倍

生産年齢人口は2025年の7,310万人から2040年には6,213万人へと約1,100万人減少する見通しで、自治体の行政サービス維持が困難になることが懸念されている。

### 業種別の労働力不足率（2040年予測）

| 職種 | 不足率 |
|------|--------|
| 介護サービス | 25.3% |
| 商品販売 | 24.8% |
| ドライバー（輸送・運搬） | 24.2% |
| 建築・土木 | 22.0% |
| 医療従事者 | 17.5% |

加えて、高度経済成長期以降に整備されたインフラ・公共施設が一斉に老朽化する問題も重なる。

## 2. 増田レポートと「消滅可能性都市」

### 2014年の衝撃

2014年5月、日本創成会議（座長：増田寛也・元総務大臣）が「消滅可能性都市」リストを発表した。

- **判定基準**: 2010年→2040年に**20〜39歳の若年女性人口が50%以上減少**する自治体
- **結果**: 全国1,799市区町村のうち**896自治体（約49.8%）**が該当
- うち**523自治体**は人口が1万人未満に転落する見込み
- 東京23区では唯一**豊島区**が該当

同年8月に増田寛也著『地方消滅』（中公新書）として出版されベストセラーとなり、政府の「地方創生」政策（まち・ひと・しごと創生法の制定、地方創生担当大臣の新設）の契機となった。

### 2024年の更新レポート

2024年4月24日、人口戦略会議（議長：三村明夫、副議長：増田寛也）が「令和6年・地方自治体『持続可能性』分析レポート」を発表。国立社会保障・人口問題研究所の最新推計（2023年12月公表）に基づく。

| 分類 | 該当数 | 概要 |
|------|--------|------|
| **自立持続可能性自治体** | **65** | 若年女性の移動減・自然減ともに20%未満 |
| **ブラックホール型自治体** | **25** | 他地域から人口を吸収するが出生率が低い（東京23区中16区含む） |
| **消滅可能性自治体** | **744** | 若年女性人口が50%以上減少見込み |
| その他自治体 | **895** | 上記以外 |

「**ブラックホール型**」は東京都心部が典型で、若者を吸い込む一方で子どもが生まれない構造的問題を示す概念として新たに注目された。

### 地域別の消滅可能性自治体数

| 地域 | 該当数 |
|------|--------|
| 東北地方 | 165（最多） |
| 北海道 | 117 |
| 九州・沖縄 | 76（最少） |

## 3. なぜ豊島区が注目されたのか

豊島区は2014年、**東京23区で唯一**「消滅可能性都市」に指定され、大きな衝撃を与えた。

### 豊島区の構造的特徴

| 指標 | 数値 | 備考 |
|------|------|------|
| 人口密度 | 約21,530人/km² | **全国市区町村1位** |
| 単身世帯比率 | **64%**（117,608世帯） | ファミリー世帯はわずか21% |
| 合計特殊出生率 | **0.81** | 極めて低水準 |
| 外国籍住民 | 11.2%（約32,732人） | — |
| 65歳以上の一人暮らし率 | **33.8%** | 全国平均17.7%の約2倍、**全国1位** |
| 高齢単身者の民営借家居住率 | **42.0%** | 23区平均32.8%を大きく上回る |
| 1住宅当たり延べ面積 | 23区中21位 | ワンルーム・単身者向けが大半 |

### 消滅可能性の根拠データ

| 指標 | 数値 |
|------|------|
| 2010年の若年女性人口（20〜39歳） | 50,136人 |
| 2040年の推計 | 24,666人 |
| **減少率** | **50.8%** |

### 問題の本質

豊島区は「若い単身者が流入するが定着せず、結婚・出産を経ずに転出し、残った住民が高齢単身者になっていく」という構造を抱えていた。人口密度は全国1位でありながら、その人口構成は極めて偏っており、持続可能性に重大な懸念があった。

## 4. 豊島区の対策 ― 危機をチャンスに

### 緊急対策本部の設置

指定直後の2014年5月、区長を本部長とする「**豊島区消滅可能性都市緊急対策本部**」を設置。2015年度に**約8,800万円**を予算化し、4つの柱で11事業を開始した。

**4つの柱：**

1. **子どもと女性にやさしいまちづくり**（最重点政策）
2. 高齢になっても元気で住み続けられるまち
3. 様々な地域との共生
4. 日本の推進力

---

### 4-A. 子育て支援・女性活躍推進

#### 組織体制

- 2016年に「**女性にやさしいまちづくり担当課長**」を民間から公募・採用し、専門部署を設置

#### 主な施策

| 施策 | 内容 |
|------|------|
| **待機児童ゼロ** | 2017年4月達成。保育施設数：52か所（2014年）→115か所（2024年）。私立認可保育所は9園→69園に拡大 |
| **夜7時までの学童保育** | 全小中学校で実施 |
| **としま鬼子母神プロジェクト** | 妊娠前のライフプラン形成支援から出産・子育てまで切れ目のない支援 |
| **マイほいくえん事業** | 未就園児家庭が区立保育園を「かかりつけ園」として利用可能 |
| **としまF1会議・100人女子会** | 若年女性の声を直接政策に反映 |
| **出産・子育て応援給付金** | 妊娠期から子育て期の各段階でギフトカードを贈呈 |
| **給食費無償化** | 子育て世帯の経済的負担軽減 |

#### 評価

- 2022年版「共働き子育てしやすい街ランキング」（日経xwoman）で**総合1位**を獲得

---

### 4-B. 公園を核としたまちづくり

「**公園がまちを変える**」コンセプトで、池袋駅周辺の4公園を順次リニューアル。

| 公園名 | リニューアル年 | 特徴 |
|--------|--------------|------|
| **南池袋公園** | 2016年 | 芝生広場とカフェを備えた憩いの場 |
| **中池袋公園** | 2019年 | Hareza池袋に隣接、アニメ文化の発信拠点 |
| **池袋西口公園** | 2019年 | 野外劇場「GLOBAL RING THEATRE」を整備 |
| **イケ・サンパーク** | 2020年 | 防災公園機能を持つ大規模緑地（グッドデザイン賞受賞） |

4公園を結ぶ電気バス「**IKEBUS（イケバス）**」が2019年から運行を開始し、池袋全体の回遊性を向上させている。

---

### 4-C. 国際アート・カルチャー都市構想

2015年に「豊島区国際アート・カルチャー都市構想」を策定。2016年に基本コンセプトを「**まち全体が舞台の誰もが主役になれる劇場都市**」と定めた。

#### 主要プロジェクト

| プロジェクト | 概要 |
|------------|------|
| **Hareza（ハレザ）池袋** | 2020年グランドオープン。ブリリアホール等を含む複合文化施設。構想のシンボル拠点 |
| **トキワ荘マンガミュージアム** | 2020年7月開館。手塚治虫・藤子不二雄らが青春時代を過ごした伝説のアパートを忠実に復元 |
| **東アジア文化都市2019豊島** | 日中韓3カ国の文化交流事業として豊島区が日本代表都市に選定 |
| **南長崎マンガランド事業** | マンガによる地域回遊型まちづくり。2024年に「トキワ荘マンガミュージアムサロン」もオープン |
| **池袋西口公園 GLOBAL RING THEATRE** | オーケストラ公演やイベントに対応する野外劇場 |

---

### 4-D. 大規模再開発（進行中）

#### 池袋駅西口地区

- エリアをA〜Dの4街区に分割
- A〜C街区に超高層複合ビル3棟、D街区に公園を整備
- 総延床面積：**約58万m²**
- 2027年度に再開発組合設立
- 2030年代から解体・新築工事開始
- **2043年の全体竣工**を目指す

#### 北デッキ

- 池袋駅の東西を結ぶ歩行者デッキ
- 2026年度に概略基本設計に着手（予算**13億2,683万円**）

## 5. 成果 ― 消滅可能性都市からの脱却

| 指標 | 2014年（指定時） | 改善後 |
|------|-----------------|--------|
| 若年女性人口減少率 | **50.8%** | **2.8%** |
| 若年女性人口（実数） | 45,520人（2014年） | 48,055人（2018年） |
| 総人口 | — | 10年間で**2万人以上増加**、40年ぶりに29万人突破 |
| 消滅可能性判定 | **該当** | **2024年4月に脱却** |

2024年4月、人口戦略会議が発表した「令和6年・地方自治体『持続可能性』分析レポート」において、豊島区は正式に消滅可能性都市から脱却した。

## 6. 残る課題と今後の展望

### 構造的課題

消滅可能性都市からの脱却は果たしたものの、以下の課題は依然として残る。

- **合計特殊出生率0.81**の低水準は改善途上
- **単身世帯64%**という偏った世帯構成
- 高齢単身者の居住支援（賃貸住宅の入居拒否問題）
- 高密度な住環境における防災・インフラ老朽化対応
- 人口流入に依存した人口維持の**持続可能性への疑問**
- 「ブラックホール型」に分類される可能性（他地域から人を吸い込むが子どもが生まれない構造）

### 将来人口推計

| 推計機関 | 2040年の豊島区人口 |
|---------|-------------------|
| 東京都推計 | 約305,682人 |
| 国立社会保障・人口問題研究所 | 約317,153人 |

### 高齢化率の推移予測

- 2020年：**19.8%**
- 2050年（推計）：**22.7%**（+2.9ポイント）

東京都全体では、2040年に高齢者世帯のうち一人暮らしの比率が**45.8%**（全国最高）に達する見込み。

### 新基本構想・基本計画（2025〜2034年度）

豊島区は新たな10年間の最上位計画を策定。理念として「みんながつながり、みんなでつくる」「だれもがいつでも主役」「『としまく』らしさがあふれる」を掲げ、8つのまちづくりの方向性を設定した。

1. 共に支えあう安全・安心なまち
2. ぬくもりのある子育てしやすいまち
3. 子ども・若者が自分らしく成長できるまち
4. 多彩な文化と産業で賑わいあふれるまち
5. 誰もが地域で共に暮らせる福祉のまち
6. 生涯にわたって健康で輝けるまち
7. 地球にも人にもやさしい持続可能なまち
8. 誰もが居心地の良い歩きたくなるまち

### SDGs未来都市

- 「SDGs未来都市」と「自治体SDGsモデル事業」のダブル選定
- 「としまSDGs都市宣言」を採択
- 第2期SDGs未来都市計画（2023〜2025年）の副題は「消滅可能性都市からの脱却〜持続して発展する『国際アート・カルチャー都市』への挑戦〜」

## 7. 他自治体との比較

| 自治体 | アプローチ | 結果 |
|--------|-----------|------|
| **豊島区（東京）** | 子育て支援＋文化＋再開発の多面的戦略 | 消滅可能性から脱却 |
| **流山市（千葉）** | 「母になるなら、流山市。」で子育て世代誘致。駅前送迎保育ステーション設置 | 自立持続可能性自治体に分類 |
| **明石市（兵庫）** | 18歳まで医療費無料、第2子以降保育料無料等「5つの無料化」 | 10年連続人口増加 |
| **奈義町（岡山）** | 人口約5,700人の小さな町で手厚い子育て支援 | 出生率全国トップクラス |
| **湯沢市（秋田）** | クラウドソーシング（在宅ワーク）の導入環境整備 | 育児・介護中の女性に就労機会創出 |

> **注意**: これらの成功例の多くは他の地域から人口を「奪う」ゼロサム的な側面もあり、日本全体の人口減少トレンドを覆すものではない点に留意が必要である。
