## 1. 豊島区の現状認識

### 人口動態の「回転ドア構造」

豊島区の最大の特徴は、若年層が大量に流入し、ライフステージの変化とともに流出する「人口の通過点」構造にある。

| 年齢層 | 社会増減（2023年） | 傾向 |
|--------|-------------------|------|
| 15〜24歳 | **+2,667人** | 進学・就職で大量転入 |
| 25〜34歳 | **-166人** | 結婚・同棲で転出開始 |
| 35歳以上 | **-1,987人** | ファミリー化で本格転出 |
| 0〜14歳 | **-550人** | 子育て世帯の流出を反映 |
| **合計** | **-36人** | わずかに転出超過 |

### 構造的特徴

| 項目 | 数値 | 備考 |
|------|------|------|
| 人口密度 | 約21,530人/km² | 全国1位 |
| 単身世帯比率 | 約75%（世帯人員1.58） | 23区2位の低さ |
| 20〜39歳の単身世帯占有率 | 48% | 23区平均38%を大幅に上回る |
| 合計特殊出生率 | 0.81〜0.89 | 極めて低水準 |
| 空き家率 | **13.94%** | **東京都で最も高い** |
| 外国人比率 | 11.73%〜12.8% | 東京都平均の約2.5倍 |
| 65歳以上の一人暮らし率 | 33.8% | 全国1位 |

### 世代別定住率（コーホート追跡）

| 年齢遷移 | 男性定住率 | 女性定住率 |
|----------|-----------|-----------|
| 20〜24歳 → 25〜29歳 | 18.2% | 20.3% |
| 25〜29歳 → 30〜34歳 | 22.7% | 25.6% |

**定住率は23区平均58.3%に対し、豊島区は約50.1%と低い。**

## 2. 「定住率」にフォーカスすることの妥当性と限界

### 妥当性：なぜ「転入増」より「転出減」が効果的か

3つの専門家エージェントが共通して認めた点：

1. **コスト効率**: 新規転入者の獲得より既存住民の定住維持の方がコストが低い（マーケティングの「1:5の法則」）
2. **財政安定性**: 定住者は安定的な税収基盤となる。短期居住者は住民登録しないケースもある
3. **社会関係資本の蓄積**: 定住者の増加が町会参加、見守り機能、災害時の共助体制を厚くする
4. **子育て環境の好循環**: ファミリー世帯の定着で保育・教育への投資判断が安定する

### 限界：批判的レビュアーからの指摘

| 論点 | 指摘内容 |
|------|---------|
| **ロジックの脆弱性** | 転居の意思決定は「住みやすさ」以外の要因（ライフステージ変化、経済的要因、勤務先変更）に大きく左右される |
| **定義の曖昧さ** | 5年コーホートか、10年か、意向調査かで数値が大きく変わる。「誰の」定住率かが不明確 |
| **逆効果のリスク** | 高齢者の定住率が上がると社会保障コスト増大→2040年問題を悪化させる可能性 |
| **出生率との乖離** | 定住率が上がっても出生率0.81が改善しなければ自然減は止まらない |
| **ゼロサム問題** | 豊島区の定住率向上が他区からの「人口の横取り」にすぎない可能性 |
| **ジェントリフィケーション** | 住みやすさ向上→地価上昇→既存低所得住民の転出という逆効果 |

### 3エージェントの合意点

> **「定住率」を唯一の目的指標にせず、セグメント別に分解し、2040年問題の解決に直結する複合指標と併用すべき。**

特に「25-39歳の子育て世帯の定住率」にフォーカスすることで、定住率向上と2040年問題解決を同時に追求できる。

## 3. 定住率が低い構造的原因

### 原因の体系化：Push-Pull-Mooring モデル

```
┌─────────────────────────────────────────────────┐
│              定住率に影響する要因                  │
├────────────┬────────────┬────────────────────────┤
│  Push要因   │  Pull要因   │  Mooring要因           │
│ (転出促進)  │ (転入促進)  │  (定住固定化)           │
├────────────┼────────────┼────────────────────────┤
│ 家賃上昇    │ 交通利便性  │ 持ち家取得             │
│ 住宅狭小    │ 商業集積    │ 子どもの学校           │
│ 子育て不安  │ 文化施設    │ 地域コミュニティ       │
│ 治安不安    │ 就業機会    │ 職場近接性             │
│ 騒音・環境  │ 医療アクセス│ 近隣の社会関係         │
│ 空き家増加  │ 子育て支援  │ 転居コスト             │
└────────────┴────────────┴────────────────────────┘
```

### 主要な構造的原因

#### (1) 住宅事情（最大の要因）

- 住宅ストックの大半がワンルーム・1K。ファミリー向け2LDK以上が絶対的に不足
- ファミリー向け物件の家賃は月額15〜20万円以上。住宅購入は5,000万円以上が標準
- **社会増減数と分譲マンション供給戸数は「極めて密接な関係」** にある（2022年に供給1,093戸→社会増減プラスに回復）
- 近隣の板橋区・練馬区・北区との価格差が転出先選択に直結

#### (2) ライフステージの断絶

- 単身→カップル→ファミリーの各段階で、区内に適切な住宅選択肢がない
- 「豊島区内でのステップアップ住み替え」が困難

#### (3) 子育て環境の課題

- 待機児童ゼロは達成済みだが、大規模公園・放課後の居場所・小児医療・教育環境に課題が残る

#### (4) コミュニティの希薄さ

- 単身世帯75%（世帯人員ベース）で「匿名的居住」が標準
- 町会・自治会加入率が低く、「根を下ろす」感覚を持ちにくい

#### (5) 高齢者の居住継続困難

- 65歳以上一人暮らしの4割が民間賃貸に居住
- 大家の高齢者入居忌避、孤独死リスク、連帯保証人問題

## 5. データ分析の具体的手法

### 分析フェーズの全体設計

```
Phase 1: 現状把握（記述統計・可視化）
    ↓
Phase 2: 要因探索（相関・回帰分析）
    ↓
Phase 3: セグメント発見（クラスター分析）
    ↓
Phase 4: トレンド予測（時系列分析）
    ↓
Phase 5: ベンチマーク（他区比較分析）
    ↓
Phase 6: 施策シミュレーション
```

### Phase 1: 記述統計・可視化

| 分析内容 | 可視化手法 | 期待されるインサイト |
|---------|-----------|-------------------|
| 年齢5歳階級別の転入転出フロー | サンキーダイアグラム | どの年齢層がどこに流出しているか |
| 町丁目別の人口増減 | 地理情報ヒートマップ（GIS） | 区内のどのエリアで定住が進み/遅れているか |
| 世帯構成別の時系列推移 | 積み上げ面グラフ | 単身→ファミリー転換の有無 |
| 転入元・転出先の地理分布 | フローマップ | 競合自治体の特定 |
| 居住期間別人口ピラミッド | 二重ピラミッド | 定住層と流動層の年齢特性 |

### Phase 2: 回帰分析

**モデル例: 町丁目レベルの横断面分析**

```
転出率_i = β0 + β1×家賃水準_i + β2×持ち家比率_i + β3×公園面積_i
           + β4×保育施設密度_i + β5×商業施設密度_i + β6×犯罪発生率_i
           + β7×外国人比率_i + β8×空き家率_i + ε_i

  i = 町丁目（約120地点）
```

### Phase 3: クラスター分析

町丁目を類型化し、エリア別の施策差別化を実現。

| 想定クラスター | 典型エリア | 施策の方向性 |
|--------------|-----------|-------------|
| A: 若年単身流動型 | 池袋駅周辺 | ステップアップ住み替え支援 |
| B: 高齢定住型 | 巣鴨・駒込周辺 | 居住継続支援・見守り強化 |
| C: ファミリー安定型 | 目白・千早周辺 | 教育環境の充実 |
| D: 外国人集住型 | 特定エリア | 多文化共生支援 |

### Phase 4: 時系列分析・コーホート分析

- ARIMA/SARIMAモデルによる月次転入転出の予測
- 施策導入前後の構造変化点検出
- **同一出生コーホートの豊島区内残存率追跡**→「離脱が起きやすい年齢」の精密特定

### Phase 5: 他区との比較分析

**比較対象区:**
- 隣接区: 新宿区、文京区、北区、板橋区、練馬区、中野区
- 類似特性区: 中野区（単身世帯率高）、渋谷区（商業・文化集積）
- ベストプラクティス区: 江東区・品川区（再開発によるファミリー層定着に成功）

### 「住みやすさ」の定量化フレームワーク

```
住みやすさ総合スコア = Σ(Wi × Di)

D1: 住宅環境スコア（家賃対所得比率、ファミリー向け住宅供給量、空き家率、住宅面積）
D2: 子育て環境スコア（保育所定員、子ども1人あたり公園面積、学力指標、支援施設数）
D3: 安全・安心スコア（犯罪発生率、交通事故、防災対策）
D4: 利便性スコア（商業施設、医療施設、交通利便性）
D5: コミュニティスコア（町会加入率、ボランティア参加率、地域イベント数）
D6: 文化・アメニティスコア（文化施設数、緑被率、公園面積率）

Wi = 住民アンケートによる重要度評価 or 主成分分析で客観的に決定
```

## 6. ターゲットセグメント別の施策提案

### 6-1. 若年単身者（20〜30代）― 定住率18〜25%

**このセグメントが最もレバレッジが大きい。** 「将来のファミリー世帯候補」であり、この層を区内に留めることが2040年問題の核心に直結する。

| 施策 | 内容 |
|------|------|
| **区内ステップアップ住み替え支援** | 単身→カップル→ファミリーと世帯が変化しても区内に住み替えられるよう、初期費用助成・ファミリー向け住宅ストック拡充 |
| **空き家リノベーション活用** | 空き家率13.94%（都内最高）を逆手に取り、ファミリー向けシェアハウス・子育て支援付き住宅・テレワーク併用住宅に転換 |
| **シビックプライドの醸成** | 住民限定の文化イベント、区民割引など、「住民であること」の付加価値を設計 |
| **サードプレイスの整備** | コワーキングスペース、コミュニティカフェ等、自宅・職場以外の「居場所」を増やし緩やかなコミュニティ形成を促す |

### 6-2. 子育て世帯 ― ファミリー世帯比率21%

| 施策 | 内容 |
|------|------|
| **ファミリー向け住宅の誘導策強化** | 新築マンション開発時に3LDK以上の一定比率確保を義務化/誘導（ワンルームマンション規制条例の強化版） |
| **子育てファミリー家賃助成の拡充** | 現行月額最大15,000円程度の増額・対象拡大 |
| **小さな公園ネットワーク** | ポケットパーク・屋上公園・高架下空間を活用した「小規模だが高頻度アクセス可能な遊び場」のネットワーク構築 |
| **公立教育の特色化** | STEAM教育、英語教育、ICT活用等で「豊島区の教育を受けたいから住む」という動機を創出 |

### 6-3. 高齢単身者 ― 一人暮らし率33.8%（全国1位）

| 施策 | 内容 |
|------|------|
| **居住支援の拡充** | 家主向け保証制度（孤独死保険の自治体負担等）の強化 |
| **共生型住宅の推進** | 空き家を活用した多世代シェアハウス型住宅 |
| **テクノロジー見守り** | IoTセンサー、見守りアプリ、AI異常検知の普及 |
| **「備えてあんしん（はれやか）」の拡充** | 身寄りのない高齢者向けの入退院支援、死後事務委任等 |

### 6-4. 外国籍住民 ― 全人口の11.73〜12.8%

| 施策 | 内容 |
|------|------|
| **22言語対応相談窓口の利用データ分析** | ニーズの高い分野に重点的にリソース配分 |
| **外国人住民の定住意向調査** | 国籍別・在留資格別の定住期間・障壁の把握 |
| **「支援対象」から「共創パートナー」へ** | 多文化コミュニティの形成を「国際アート・カルチャー都市」との相乗効果に |

## 7. 先行事例

### 7-1. 高槻市（大阪府）― EBPMによる定住促進プロモーション

**最も参考になる事例。**

- 転出入者アンケートと住民基本台帳データを統計分析
- 20〜30代のDINKS・カップル層を「準ファミリー層」と定義
- ターゲットに交通広告・Web広告をピンポイント配信
- **結果: 令和2〜3年連続で20〜40代が転入超過に改善**

**豊島区への示唆**: 転出届提出時のアンケートを体系的に実施・分析することが最優先。

### 7-2. 流山市（千葉県）― マーケティング戦略

- 自治体初のマーケティング課を設置
- 「母になるなら、流山市。」で共働き子育て世帯（DEWKS）をターゲティング
- **結果: 人口15.2万人→20.5万人、出生率1.14→1.55に向上**

**豊島区への示唆**: 流山市は「郊外×都心アクセス」が武器。豊島区は逆に「都心なのに住み続けられる」を打ち出す必要がある。

### 7-3. 会津若松市（福島県）― データ駆動型スマートシティ

- 「公・民・学連携」+「データ駆動」で地域経営
- 行政データを横断分析し、サービスを最適設計

**豊島区への示唆**: 転出リスクの高い世帯を早期特定する「予測モデル」構築の可能性。

### 7-4. 飯南町（島根県）― 長期居住インセンティブ

- 月4万円の家賃を25年間支払えば住宅を無償取得できる定住住宅
- 29棟に115人（うち子ども54人）が入居

**豊島区への示唆**: 都市部では直接適用困難だが、「10年以上居住世帯への住民税控除」「区営住宅優先入居」等の長期居住インセンティブは応用可能。

## 8. 批判的検討：このアプローチの弱点と改善策

### 8-1. ロジックの脆弱性

| 問題 | 詳細 | 改善策 |
|------|------|--------|
| **住みやすくても出ていく** | ライフステージ変化は住みやすさとは無関係に転居を引き起こす。30代は23区全体で転出超過 | 「住みやすさ」ではなく「住み続けやすさ（Affordability）」に焦点を移す |
| **自己矛盾** | 住みやすさ向上施策（公園整備、再開発）→地価・家賃上昇→既存住民の転出を促進 | ジェントリフィケーション対策を同時に設計する |
| **行政の影響力の限界** | J-STAGE研究で「行政サービスと定住率の相関は弱い」との結果。家賃・住宅は市場原理で決まる | 行政がコントロール可能な変数を明確にし、そこに集中する |

### 8-2. データアプローチの限界

| 問題 | 詳細 | 改善策 |
|------|------|--------|
| **相関≠因果** | 「公園が多い→定住率高い」は、持ち家率が高い住宅地の特性かもしれない | 差分の差分法（DID）、回帰不連続デザイン（RDD）等で因果推定 |
| **転出者の声が欠落** | 住み続けている人への調査では選択バイアスが生じる | **転出者調査を最優先で実施** |
| **データで見えないもの** | 帰属意識、将来への期待、個人のライフプラン、コミュニティの質 | 質的調査（インタビュー・ワークショップ）との併用 |
| **手元データへの引きずられ** | オープンデータは分析しやすいが、定住率を動かすレバーの個人レベルデータは入手困難 | **仮説駆動型**：「なぜ30代が出ていくか？」から出発し、必要データを設計 |

### 8-3. 豊島区特有のリスク

| リスク | 詳細 |
|--------|------|
| **ブラックホール型の矛盾** | 出生率0.81のまま人口を「囲い込む」ことは、国全体の人口問題を悪化させるという批判がありうる |
| **単身世帯の「定住」は現実的か** | 持ち家率低、転居コスト低、条件差で容易に流動。単身者の住みやすさは「既に豊島区が強い分野」に集中しており改善余地が小さい |
| **再開発のパラドックス** | 街がきれいに便利になること ≠ 今の住民が住み続けられること |

### 8-4. 改善のための5つの提言

> 1. **「定住率」を唯一の指標にしない** → 若年女性人口、出生率、世帯構成変化を併用
> 2. **「誰の」定住率かを明確にする** → 「25-39歳の子育て世帯の定住率」にフォーカス
> 3. **「住みやすさ向上」と「住み続けやすさの確保」を区別する** → 再開発が前者に偏ると後者を毀損する
> 4. **仮説駆動でデータを設計する** → 「手元データで何が言えるか」ではなく「仮説検証に何が必要か」
> 5. **転出者の分析を最優先する** → 住み続けている人より、出て行った人の声にヒントがある

## 9. 統合提言：推奨アクションプラン

### KPI設計

#### 最終KPI（3-5年で評価）

| KPI | 目標 | 測定方法 |
|-----|------|---------|
| 25-39歳の転出超過率 | 50%削減 | 住民基本台帳人口移動報告 |
| 5年以上居住者割合 | +5ポイント | 国勢調査（2030年） |
| ファミリー世帯の純社会増 | ±0（均衡） | 世帯構成別転入転出データ |

#### 中間KPI（年次で評価）

| KPI | 目標方向 |
|-----|---------|
| ファミリー向け住宅供給戸数 | 増加 |
| 空き家活用件数 | 増加（空き家率低下） |
| 子育て世帯家賃助成利用数 | 増加 |
| 住民定住意向率 | 80%以上 |

### 分析の優先順位

| 優先度 | 分析タスク | 理由 |
|-------|-----------|------|
| **最優先** | 年齢層別×世帯構成別の転入転出フロー分析 | 「誰が、どこに出て行くか」が全施策の起点 |
| **最優先** | 転出先自治体との比較分析（家賃・住宅面積・子育て指標） | 転出理由の定量的把握 |
| **高** | 町丁目別クラスター分析と地域別定住率の差異 | エリア別施策の差別化 |
| **高** | 空き家の地理分布と活用可能性 | 豊島区最大の「資産」（13.94%）の活用 |
| **中** | コーホート追跡による離脱タイミング分析 | 予防的介入ポイントの特定 |
| **中** | 住民意識調査データとの統合分析 | 定量では見えない「主観的住みやすさ」 |

### 施策の優先順位

| 時間軸 | 施策 |
|--------|------|
| **即時着手** | 転出者アンケートの体系化と分析基盤構築（高槻市モデル） |
| **即時着手** | セグメント別定住率の定期モニタリング体制確立 |
| **短期（1-2年）** | ファミリー向け住宅の戦略的誘導 |
| **短期（1-2年）** | 区内住み替え支援制度の創設 |
| **短期（1-2年）** | 高齢者居住支援の拡充 |
| **中期（3-5年）** | 公立教育の特色化 |
| **中期（3-5年）** | 外国籍住民の定住促進策の体系化 |
| **中期（3-5年）** | データ基盤の横断統合と予測モデル構築 |

### 核心戦略：空き家率都内最高を最大の政策レバーに

豊島区の空き家率13.94%は「課題」であると同時に「最大の機会」である。

```
空き家1,000戸をファミリー向けにリノベーション
    → 約2,500人のファミリー定着（推計）
    → 転出超過率を約30%改善
    → 空き家率低下と定住率向上を同時に達成
```
