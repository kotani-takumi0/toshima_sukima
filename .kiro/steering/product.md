# Product Overview

豊島区内の高校生・大学生が、空き時間に地域の小さな困りごとを解決できる短時間マッチングサービス。
単発・短時間の依頼に対して、学生が安全に応募し、地域との接点を作ることを目的とする。

## Core Capabilities

- 豊島区内の学生のみ利用可能な本人属性チェック
- 豊島区内の個人事業主・地域施設のみ投稿可能な事業者チェック
- 30〜60分中心の「お手伝い」募集作成・公開・応募
- 現金報酬と非金銭報酬（おもてなし）の両対応
- 応募から採用確定までの簡易フロー

## Target Use Cases

- 学生が空きコマや通学の前後に短時間の地域タスクを行う
- 子育て世帯がコミュニティスペースでの短時間見守りを依頼する
- 高齢者がスマホ操作サポートを依頼する
- 外国人店主がメニュー日本語校正などを依頼する
- 移動補助（買い物袋運搬、階段でのベビーカー補助）を依頼する

## Value Proposition

- 学生にとっては「短時間で地域とつながる」選択肢を提供
- 地域事業者・住民にとっては「すぐ頼める身近な支援」を提供
- 報酬を金銭に限定しないことで、地域文化や店舗体験を価値化できる
